## 一、玄學語境下的題目設計原則

### 1.1 「去問卷化」原則 (De-survey Principle)

- **儀式感優先**：使用者的互動體驗應被感知為「選象、取兆、應機」，而非填寫醫療問卷或心理測驗。
- **單一物件選擇**：避免滑桿、李克特量表與多選題；優先採用單選、直覺式決定。
- **象徵化互動**：選項應以符號、元素、狀態或情境呈現，模擬抽籤、擲卦、選牌。

### 1.2 題目語境要求 (Contextual Purity)

- **詞彙限制**：題目與選項必須僅使用 `vocabulary_metaphysical.v1.0.json` 中的詞彙。
- **嚴禁外顯語境**：不得出現任何心理學、科學、醫療或診斷相關用語。
- **正反示例**：
  - ❌ 錯誤：你最近是否感到焦慮？
  - ✅ 正確：近來是否常覺心神不寧，氣滯難行？

### 1.3 題目結構要求 (Structural Mapping)

- **Stage 1（八卦錨定）**：題目引導使用者從八個具象象徵（天、地、雷、風、水、火、山、澤）中選擇其一，對應 `bagua_8_trigrams.v1.0.json`。
- **Stage 2（五行狀態）**：題目描述五種能量狀態（生長、顯化、穩定、收斂、流動），對應 `wuxing_5_elements.v1.0.json`。
- **Stage 3（深度剖析）**：題目負責觸發強度權重（Intensity Score），其結果作為 Hexagram Deriver 變爻計算的唯一輸入。

## 二、題目與系統的對應關係

### 2.1 題目 → 玄學運算對應

- **Upper Trigram（外顯狀態）**：
  - 題目語境：外部環境、當下局勢、顯性事件。
  - 對應方式：選項直接映射至 Upper Trigram ID。
- **Lower Trigram（內在狀態）**：
  - 題目語境：內在心境、隱性趨勢、內部動能。
  - 對應方式：選項直接映射至 Lower Trigram ID。
- **Variable Line（變爻觸發）**：
  - 題目語境：事件急迫性、變化幅度、張力強弱。
  - 對應方式：單選結果轉換為 0–100 權重值，輸入至 `calculate_variable_line`。

### 2.2 題目 → 科學系統對應（隱藏層）

- 題目選項在後端皆綁定心理學構念與風險指標，僅供內部計算與驗證。
- 玄學層之「吉 / 凶 / 阻 / 通」必須與後端風險等級保持一對一一致。

## 三、題目設計指南

### 3.1 最佳實踐流程

1. 確定後端欲觀測之心理或風險指標。
2. 於 `vocabulary_psychology_mapping.v1.0.json` 中找到對應之玄學概念。
3. 使用玄學詞彙庫撰寫題目與選項。
4. 進行禁用詞檢查與對應關係驗證。

### 3.2 常見錯誤

- 混入心理或科學語境（如「壓力」「認知」）。
- 使用傳統問卷形式（滑桿、多選）。
- 題目語義無法對應至後端運算欄位。

### 3.3 驗收標準

- 題目文本不含任何禁用詞。
- 每一選項皆可映射至明確後端輸入值。
- 語境完整，符合玄學敘事與儀式感。
