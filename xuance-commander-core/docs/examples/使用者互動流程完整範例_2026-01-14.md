### Stage 3：V3 引擎計算

**輸入數據**：
```python
answers = [3, 2, 2, 3, 2, 3, 2, 2]
# Q1=3, Q2=2, Q3=2, Q4=3, Q5=2, Q6=3, Q7=2, Q8=2
```

**計算過程**：

#### 步驟 1：計算各維度平均分
```python
# 核心狀態（Q1-Q2）
core_avg = (3 + 2) / 2 = 2.5

# 反應模式（Q3-Q4）
reaction_avg = (2 + 3) / 2 = 2.5

# 外顯行為（Q5-Q6）
manifestation_avg = (2 + 3) / 2 = 2.5

# 能動性（Q7-Q8）
agency_avg = (2 + 2) / 2 = 2.0
```

#### 步驟 2：計算 Severity（加權平均）
```python
# 權重：核心 40%，反應 30%，外顯 20%，能動性 10%
severity_raw = (2.5 * 0.4) + (2.5 * 0.3) + (2.5 * 0.2) + (2.0 * 0.1)
             = 1.0 + 0.75 + 0.5 + 0.2
             = 2.45

# 標準化到 0-1
severity = 2.45 / 4 = 0.6125 ≈ 0.61
```

#### 步驟 3：判定 Rigidity State
```python
if severity >= 0.75:
    rigidity = "Critical"
elif severity >= 0.55:
    rigidity = "Locked"
else:
    rigidity = "Fluid"

# 本例：0.61 >= 0.55 → "Locked"（鎖定）
```

#### 步驟 4：計算 Agency Score
```python
agency_score = 1 - (agency_avg / 4)
             = 1 - (2.0 / 4)
             = 1 - 0.5
             = 0.5
```

#### 步驟 5：判定 Safety Mode
```python
if severity >= 0.75:
    safety_mode = "L4"
elif severity >= 0.55:
    safety_mode = "L3"
elif severity >= 0.25:
    safety_mode = "L2"
else:
    safety_mode = "L1"

# 本例：0.61 >= 0.55 → "L3"（中高風險）
```

#### 步驟 6：選擇卦象
```python
hexagram = select_hexagram(
    rigidity="Locked",
    domain="Zhen",
    severity=0.61
)

# 基於 Locked + Zhen + mid-high severity
# 選擇：hexagram_51_shock（震為雷 - 重雷震）
```

#### 步驟 7：判定五行失衡
