## 📋 三層互動流程

### 流程 1：八卦門（符碼映射）

**畫面**：
```
┌──────────────────────────────┐
│     【起心動念】              │
│                              │
│   請在以下辭象中選擇         │
│   最能描述你當下感受的字     │
│                              │
│   困  擾  艱  滯              │
│   燃  散  衝  穩              │
│                              │
│   （點擊一個字）              │
└──────────────────────────────┘
```

**動畫**：選擇後，八卦之門緩慢浮現 ☵

### 流程 3：萬象納甲（辭象凝視）

**畫面**：
```
┌──────────────────────────────┐
│     【萬象顯化】              │
│                              │
│   坎水既定，事在妻財。        │
│                              │
│   你靜立於廳堂之中，          │
│   四周緩緩浮現數道辭象：      │
│                              │
│   懼   責   內   嚴          │
│   空   壓   罪   逃          │
│   冷   困   依   撐          │
│                              │
│   請凝視那些觸動你的辭象      │
│   （已選 3 個，可選 3-5 個）   │
│                              │
│   已選：懼 內 嚴             │
│                              │
│   [確認]                     │
└──────────────────────────────┘
```

**核心交互**：
- 辭象逐一浮現（0.5 秒間隔）
- 點擊辭象 → 放大 + 發光
- 可以取消選擇（再次點擊）
- 最少選 3 個，最多選 5 個
