## ✅ P2-1：年齡適應版本

### 任務概述

**檔案位置**：`P2-1_年齡適應版本_2026-01-13.md`

**目標**：建立青少年版（13-19 歲）和老年版（60+ 歲），實現全年齡覆蓋

**內容**：
- 青少年版：簡化語言、現代活動、校園情境
- 老年版：白話語言、靜態活動、退休情境

### 核心概念

**當前問題**：
- 語言風格偏文言（「您如龍行於天」）→ 青少年理解困難
- 活動形式偏成年（30-60 分鐘健行）→ 老年人體力負擔
- 情境描述偏職場（工作壓力、資源競爭）→ 青少年／老年不適用

**調整方向**：
- 青少年：「你」、現代比喻、10-20 分鐘、校園情境
- 老年：白話文、低強度活動、15 分鐘、退休情境

### 交付物

**總計**：16 個檔案
- 青少年版：8 個檔案
- 老年版：8 個檔案

### 預估時間

**14-21 天**
- 第 1-2 天：熟悉背景
- 第 3-8 天：青少年版
- 第 9-14 天：老年版
- 第 15-21 天：測試與優化

## ✅ P2-2：文化適應版本

### 任務概述

**檔案位置**：`P2-2_文化適應版本_2026-01-13.md`

**目標**：建立文化中性版本和註釋說明版本，擴展到全球市場

**內容**：
- 文化中性版：去除強烈東方符號，使用普世隱喻
- 註釋說明版：解釋玄學概念，提供文化背景

### 核心概念

**當前問題**：
- 強烈東方文化（龍、鳳凰、五行、八卦）→ 西方用戶陌生
- 文言美學（「亢龍之憂」）→ 語言障礙
- 東亞情境（孝道、面子）→ 價值觀差異

**調整方向**：
- 文化中性：龍→風暴、五行→元素、孝道→尊重
- 註釋版：解釋玄學概念、提供文化背景、多語言支援

### 交付物

**總計**：18 個檔案
- 文化中性版：9 個檔案
- 註釋說明版：9 個檔案

### 預估時間

**21-28 天**
- 第 1-3 天：熟悉背景
- 第 4-14 天：文化中性版
- 第 15-26 天：註釋說明版（含多語言）
- 第 27-28 天：測試與優化

## ✅ P2-3：動態追問系統

### 任務概述

**檔案位置**：`P2-3_動態追問系統_2026-01-13.md`

**目標**：建立智能追問系統，根據初步分數動態追加題目，提升診斷精準度

**內容**：
- 追問觸發邏輯（3 種策略）
- 追問題庫（150-225 題）
- 演算法設計（觸發 + 調整）

### 核心概念

**當前問題**：
- 固定 8 題（OCTET_8）→ 可能不足以精準診斷
- 高風險區（≥ 0.7）→ 需要更深入探測
- 模糊區（0.4-0.6）→ 難以判斷真實狀態
- 邊界狀態（接近 threshold）→ 容易誤判

**調整方向**：
- 深度探測：持續性、多領域、恢復能力（high 區）
- 區辨題：頻率、強度、行為影響（模糊區）
- 邊界確認：對比題（邊界區）

### 交付物

**總計**：10 個檔案
- 觸發邏輯：3 個檔案
- 追問題庫：5 個檔案
- 演算法設計：2 個檔案

### 預估時間

**14-21 天**
- 第 1-3 天：熟悉背景
- 第 4-7 天：觸發邏輯
- 第 8-15 天：追問題庫
- 第 16-19 天：演算法設計
- 第 20-21 天：測試與優化
