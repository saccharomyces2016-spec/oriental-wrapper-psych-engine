## 🎯 P2-1：年齡適應版本

### 用一個例子理解

**成人版**（當前，適用 20-60 歲）：
```
診斷敘事：「您如龍行於天，氣勢充盈，然此刻陽剛過盛，恐有亢龍之憂。」
活動建議：「每週進行一次戶外健行，至少 1 小時，讓木氣在陽光下充分舒展。」
```

**青少年版**（P2-1 目標，適用 13-19 歲）：
```
診斷敘事：「你現在的能量就像火焰全開，燃燒得很旺。但火燒太猛，會把自己也燒乾。」
活動建議：「每天花 15 分鐘做你喜歡的運動（籃球、跑步都可以），讓過旺的火氣自然散去。」
```

**老年版**（P2-1 目標，適用 60+ 歲）：
```
診斷敘事：「您現在的狀態就像火燒得很旺。雖然有衝勁，但用力過度，反而容易累。」
活動建議：「每天早上或傍晚，到公園散步 15-20 分鐘，讓能量慢慢流動起來。」
```

**效果**：不同年齡的人都能理解並共鳴

## 🎯 P2-2：文化適應版本

### 用一個例子理解

**東方版**（當前，適用東亞文化）：
```
「您如龍行於天，氣勢充盈，然此刻陽剛過盛，恐有亢龍之憂。」
「建議您採『金生水』之法，將過盛的剛健之氣，轉化為深沉的智慧儲藏。」
```

**文化中性版**（P2-2 目標，適用全球）：
```
「你現在的能量就像一場風暴，力量強大且充滿氣勢。但風暴若不停歇，最終會耗盡自身。」
「建議你將強烈的能量，轉化為深層的智慧累積，就像將火的熱度，轉變為溫暖的光芒。」
```

**註釋版**（P2-2 目標，為不熟悉玄學的用戶）：
```
「您如龍行於天，氣勢充盈，然此刻陽剛過盛，恐有亢龍之憂。」
[註釋：「龍」在東方文化中象徵強大的能量和領導力。「陽剛過盛」意味著力量過度，需要平衡。]
```

**效果**：西方用戶也能理解並共鳴

## 🎯 P2-3：動態追問系統

### 用一個例子理解

**當前流程**（靜態，固定 8 題）：
```
1. 用戶回答 8 題 OCTET_8
2. 計算 Severity = 0.75（high）
3. 直接呈現結果
```
→ 可能誤判（短期疲勞 vs 真正倦怠）

**P2-3 流程**（動態，智能追問）：
```
1. 用戶回答 8 題 OCTET_8
2. 計算 Severity = 0.75（high）
3. 系統判斷：「需要深度探測」
4. 追加 2-3 題：
   - 「這種狀態持續多久了？」
   - 「是否影響多個生活領域？」
   - 「休息後會改善嗎？」
5. 根據追問答案調整 Severity
6. 呈現最終結果
```
→ 診斷更精準

**效果**：診斷準確度從 80-85% → 90-95%

## 💡 記住這 5 個關鍵

### 1. 100% 玄學語境

**✅ 正確**：
- 能量、氣場、流動、平衡
- 火焰、風暴、河流、山（自然元素）
- 狀態、節奏、趨勢、局勢

**❌ 錯誤**：
- 焦慮、憂鬱、壓力、緊張
- 治療、症狀、疾病、診斷

### 2. 適應但不失核心

**✅ 正確**：
- 青少年版：簡化語言，但仍是玄學語境
- 文化中性：去除東方符號，但仍用自然隱喻
- 動態追問：智能化，但仍是玄學題目

**❌ 錯誤**：
- 變成心靈雞湯
- 變成心理學問卷
- 失去玄學美學

### 3. 測試導向

**P2 系列需要測試**：
- 青少年測試：找 13-19 歲測試者
- 老年測試：找 60+ 歲測試者
- 跨文化測試：找西方背景測試者

**目標**：80%+ 表示「能理解」「有共鳴」

### 4. 不刻板化

**✅ 正確**：
- 青少年：理解他們的困擾（考試、社交、未來）
- 老年：理解他們的智慧（經驗、價值、角色）
- 西方：理解普世性需求（不是東方 vs 西方）

**❌ 錯誤**：
- 青少年 = 叛逆、膚淺
- 老年 = 固執、落伍
- 西方 = 不懂玄學

### 5. 保持深度

**P2 是優化，不是降級**：
- ✅ 簡化語言，但保持洞察深度
- ✅ 調整情境，但保持共鳴感
- ✅ 文化中性，但保持玄學美學
