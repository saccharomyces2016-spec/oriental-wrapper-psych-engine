## ⏳ P1-2：動態敘事組合引擎

### 任務概述

**檔案位置**：`P1-2_動態敘事組合引擎_2026-01-13.md`

**目標**：建立動態敘事組合引擎，讓卦象敘事根據使用者的個人情境（六親物象 Context）動態調整，實現「準到發毛」的個人化體驗。

**內容**：
- Context 插入點設計（64 個卦象 × 5-10 個插入點）
- 敘事片段庫建立（5 個 Context × 10-15 個片段 = 50-75 個片段）
- 組合規則引擎（動態組合邏輯）

### 核心概念

**什麼是六親物象（Context）？**

| 類別 | 中文 | 英文 | 象徵 | 範例情境 |
|------|------|------|------|---------|
| **印** | Authority | Authority | 長輩、導師、規則 | 父母、老師、上司、社會規範 |
| **官** | Power | Power | 權威、壓力、責任 | 工作壓力、法律義務、社會期待 |
| **財** | Resource | Resource | 資源、物質、關係 | 金錢、時間、人脈、機會 |
| **比** | Peer | Peer | 同儕、競爭、合作 | 朋友、同事、競爭對手 |
| **食** | Output | Output | 創造、表達、產出 | 創作、溝通、影響力、成就 |

**為什麼需要動態組合？**

**通用敘事**（當前）：
```
您如龍行於天，氣勢充盈，然此刻陽剛過盛，恐有亢龍之憂。
您的意志如磐石堅定，行動如疾風迅雷，卻忘了張弛有度之理。
```

**個人化敘事**（P1-2 目標）：
```
您如龍行於天，氣勢充盈，然此刻陽剛過盛，恐有亢龍之憂。
[插入 Authority] 您與權威的碰撞，並非來自外界的壓制，而是您內在的剛強無法順應天時。
長輩或導師的指引，您視為束縛，卻忘了真正的強者，懂得在必要時低頭。
```

**效果**：
- ✅ 從「準確」→「準到發毛」
- ✅ 從「當頭棒喝」→「被看見」
- ✅ 從「玄學包裝」→「個人化共鳴」

### 交付物

**總計**：10 個檔案

**任務 1：Context 插入點設計**（2 個檔案）
1. `hexagram_context_insertion_points.v1.0.json`

**任務 2：敘事片段庫建立**（7 個檔案）

## ⏳ P1-3：長鏈風險預測

### 任務概述

**檔案位置**：`P1-3_長鏈風險預測_2026-01-13.md`

**目標**：建立長鏈風險預測系統，為高風險 Facet 建立 3-4 階段的連鎖反應預測，讓使用者清楚看見「如果不改善，會發生什麼」。

**內容**：
- 高風險 Facet 選擇（20-30 個）
- 長鏈路徑設計（每個 Facet 2-3 條路徑，每條路徑 3-4 個 stage）
- 預防建議設計（每個 stage 具體的預防建議）

### 當前問題

**短鏈風險預測**（P0-3 已完成）：
```
"mid": {
  "if_not_improve": [
    "持續警覺 → 體力逐漸耗盡 → 進入慢性疲勞",
    "外界刺激累積 → 反應閾值下降 → 小事也能引發過激反應"
  ]
}
```

**特點**：
- ✅ 簡潔明了（2-3 個 stage）
- ⚠️ **預測深度不足**（使用者可能覺得「還不夠嚴重」）
- ⚠️ **缺少分支點**（單一路徑，缺少多種可能性）

### 長鏈風險預測（P1-3 目標）

**範例：chronic_alertness（慢性警覺）**

**路徑 1：職場崩潰路徑**
- Stage 1：持續警覺狀態 → 體力與專注力消耗
- Stage 2：體力與專注力下降 → 工作失誤開始出現
- Stage 3：工作失誤累積 → 職業危機（被質疑能力）
- Stage 4：失去工作 → 經濟與身分危機 → 家庭關係緊張

**路徑 2：健康崩潰路徑**
- Stage 1：持續警覺狀態 → 忽略身體信號
- Stage 2：身體症狀出現 → 頭痛、失眠、胃痛
- Stage 3：慢性疾病形成 → 免疫力下降
- Stage 4：嚴重疾病爆發 → 生活品質全面下降

**效果**：
- ✅ 警示性增強（4 個 stage，讓使用者看見「遠期後果」）
- ✅ 個人化共鳴（根據情境顯示不同路徑）
- ✅ 預防實用性（每個 stage 都有對應的預防建議）

### 交付物

**總計**：24-34 個檔案

**任務 1：高風險 Facet 選擇**（1 個檔案）
1. `high_risk_facets_for_longchain.v1.0.json`

**任務 2：長鏈路徑設計**（22-32 個檔案）
2-21. `longchain_*.risk.v2.0.json`（20-30 個，每個 Facet 一個）
22. `longchain_risks_complete.v2.0.json`（整合檔案）
23. `LONGCHAIN_RISKS_README.md`（文檔說明）

**任務 3：預防建議設計**
- 已包含在任務 2 中（prevention_per_stage 欄位）

### 關鍵要求

- ✅ 每個 Facet：2-3 條路徑
- ✅ 每條路徑：3-4 個 stage
- ✅ 每條路徑：至少 1 個 branch point
- ✅ 每個 stage：具體的預防建議
- ✅ 100% 玄學語境（0% 心理學/醫療詞彙）
