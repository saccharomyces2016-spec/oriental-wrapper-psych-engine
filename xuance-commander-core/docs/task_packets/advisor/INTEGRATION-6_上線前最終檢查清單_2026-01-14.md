```markdown
# 使用者體驗驗證

## 1. 真人測試

### 測試對象（15 人）
- 5 人：20-30 歲（年輕族群）
- 5 人：30-50 歲（中年族群）
- 5 人：50+ 歲（中老年族群）

### 測試問題
1. 「你覺得這是什麼？」
   - ✅ 期待答案：「算命」、「占卜」、「易經」
   - ❌ 失敗答案：「心理測驗」、「問卷調查」

2. 「你覺得準嗎？」
   - ✅ 期待：> 80% 認為「準」或「很準」

3. 「你會推薦給朋友嗎？」
   - ✅ 期待：> 70% 願意推薦

## 2. A/B 測試

### 測試項目
- 辭象 vs 直述：哪種更有玄學感？
- 簡單動畫 vs 無動畫：哪種更有質感？
- 2 秒載入 vs 5 秒載入：哪種更有神秘感？

## 3. 錯誤處理測試

### 測試場景
- 網路中斷時的 UX
- API 錯誤時的 UX
- 使用者操作錯誤時的提示
```
