### 3. 為什麼需要長鏈風險預測？

#### 3.1 當前問題

**問題 1：預測深度不足**
- 當前 riskchains：2-3 個 stage
- 使用者可能覺得「還不夠嚴重」，缺乏改變動力

**問題 2：缺少分支點**
- 當前是單一路徑（A → B → C）
- 實際上，風險會因個人情境而分支：
  - 路徑 A：A → B1 → C1 → D1（職場線）
  - 路徑 B：A → B2 → C2 → D2（家庭線）

**問題 3：缺少階段性預防**
- 當前只有最終的 prevention
- 缺少每個 stage 的階段性預防建議

#### 3.2 長鏈風險預測的價值

**價值 1：增強警示性**
- 4 個 stage 的連鎖反應，讓使用者看見「遠期後果」
- 例如：chronic_alertness（慢性警覺）
  - Stage 1：持續警覺 → 體力耗盡
  - Stage 2：慢性疲勞 → 注意力下降
  - Stage 3：工作失誤累積 → 職業危機
  - Stage 4：失去工作 → 經濟危機 → 家庭破裂

**價值 2：提供分支可能性**
- 根據使用者的情境（Context），顯示不同的風險路徑
- 例如：burnout_syndrome（倦怠症候群）
  - 路徑 A（職場線）：工作倦怠 → 績效下降 → 失去工作
  - 路徑 B（健康線）：工作倦怠 → 身體不適 → 慢性疾病
  - 路徑 C（關係線）：工作倦怠 → 情緒失控 → 人際破裂

**價值 3：提供階段性預防**
- 每個 stage 都有對應的預防建議
- 使用者可以在任何階段「剎車」

---

### 4. 長鏈風險預測的設計原則

#### 4.1 Stage 設計原則

**原則 1：邏輯合理**
- 每個 stage 必須是前一個 stage 的合理後果
- 避免跳躍性推論

**原則 2：漸進惡化**
- Stage 1 → Stage 2 → Stage 3 → Stage 4
- 每個 stage 比前一個更嚴重

**原則 3：具體可感**
- 避免抽象描述（例如「狀態惡化」）
- 使用具體情境（例如「工作失誤累積」）

**原則 4：玄學語境**
- 使用五行、氣、勢、局等詞彙
- 避免醫療/心理學詞彙

#### 4.2 Branch Point 設計原則

**原則 1：情境分支**
- 根據使用者的情境（Context），設計不同的路徑
- 例如：職場 vs 家庭 vs 健康

**原則 2：概率估算**
- 每條路徑有概率估算（high / medium / low）
- 幫助使用者判斷「這可能發生在我身上嗎」

**原則 3：路徑數量**
- 每個 Facet：2-3 條路徑
- 避免過多路徑（造成混亂）

### 2.3 設計要求

**2.3.1 Stage 設計**

**每個 Stage 必須包含**：
- ✅ **state**：當前狀態（簡短描述）
- ✅ **description**：詳細描述（30-50 字）
- ✅ **metaphor**：玄學隱喻（10-20 字）
- ✅ **energy_cost**：能量消耗程度
- ✅ **warning_signs**：警示信號（2-3 個）
- ✅ **consequences**：後果（Stage 4 必須）

**2.3.2 Branch Point 設計**

**每個 Branch Point 必須包含**：
- ✅ **at_stage**：在哪個 stage 出現分支
- ✅ **decision**：分支的決策點
- ✅ **branch_a** & **branch_b**：兩種可能性
- ✅ **outcome**：不同選擇的後果

**2.3.3 Prevention 設計**

**每個 Stage 的 Prevention 必須包含**：
- ✅ **advice**：具體的預防建議
- ✅ **why**：為什麼這個建議有效（用玄學語言）

## 🎯 任務 3：預防建議設計

### 3.1 目標

為每個 Stage 和每個 Branch Point 建立**具體可執行的預防建議**。

### 3.2 預防建議原則

**原則 1：階段性**
- 每個 Stage 的預防建議必須針對該階段
- Stage 1 的建議：輕度預防（調整生活方式）
- Stage 2 的建議：中度干預（尋求支持）
- Stage 3 的建議：重度干預（專業協助）
- Stage 4 的建議：危機處理（重建生活）

**原則 2：具體可執行**
- 避免空泛建議（例如「多休息」）
- 提供具體步驟（例如「每日睡前 10 分鐘，關閉所有電子設備」）

**原則 3：玄學語境**
- 使用五行、氣、勢、局等詞彙
- 解釋為什麼這個建議有效（用玄學語言）
