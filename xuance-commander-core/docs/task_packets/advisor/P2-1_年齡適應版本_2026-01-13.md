## 🎯 終極目標 (North Star)

**世界級產品目標**：
- **全人類適用**：All Humans（不同年齡、文化、背景）
- **全年齡覆蓋**：13-80 歲（青少年到老年）
- 提供「scary accurate」（令人驚訝的準確）的心理診斷
- 純玄學外觀 + 絕對科學內核

**本任務包目標**：
建立 **年齡適應版本（Age-Adaptive Versions）**，讓系統能根據使用者的年齡，自動調整語言風格、活動形式、敘事深度，實現真正的「全年齡覆蓋」。

### 3. 年齡適應的必要性

#### 3.1 為什麼需要年齡適應？

**問題 1：語言風格不適配**

**當前敘事**（偏文言）：
```
您如龍行於天，氣勢充盈，然此刻陽剛過盛，恐有亢龍之憂。
您的意志如磐石堅定，行動如疾風迅雷，卻忘了張弛有度之理。
```

**青少年反應**：
- 「這什麼意思？」（理解困難）
- 「太老派了」（文化距離）
- 「跟我無關」（共鳴度低）

**老年人反應**：
- 「字太文言，看不太懂」（閱讀障礙）
- 「能不能用白話文說？」（理解困難）

---

**問題 2：活動形式不適配**

**當前活動**（偏成年）：
```
「每週進行一次戶外健行，至少 1 小時，讓木氣在陽光下充分舒展」
```

**青少年反應**：
- 「健行？我寧願打球或跑步」（活動形式）
- 「1 小時太久了」（時間長度）

**老年人反應**：
- 「1 小時健行太累了」（體力負擔）
- 「我更適合散步」（活動強度）

---

**問題 3：情境描述不適配**

**當前情境**（偏成年）：
```
「您對資源的渴求，如乾涸之地渴望甘霖。金錢、機會、人脈，您想全數掌握...」
```

**青少年反應**：
- 「金錢？人脈？我的困擾是考試和同學關係」（情境不符）

**老年人反應**：
- 「我已經退休了，資源問題不是我的重點」（情境不符）

---

#### 3.2 年齡適應的價值

**價值 1：提升共鳴度**
- 青少年版：使用現代語言、校園情境、社交網絡
- 老年版：使用白話語言、退休情境、健康關注

**價值 2：擴大用戶群**
- 從「20-60 歲」→「13-80 歲」
- 市場規模擴大 3-4 倍

**價值 3：真正達成「全人類」目標**
- 不再受限於特定年齡層
- 符合終極目標：All Humans, All Ages, All Issues

## 🎯 任務 1：青少年版本（13-19 歲）

### 1.1 語言風格調整

#### 調整原則

**原則 1：簡化文言**
- 減少古文用語
- 使用現代白話
- 保持玄學核心

**原則 2：增加現代元素**
- 使用青少年熟悉的比喻
- 融入現代科技、流行文化
- 保持不俗氣

**原則 3：語氣親近**
- 從「您」→「你」
- 語氣更輕鬆、不說教
- 理解而非教訓

#### 範例對比

**成人版**（當前）：
```json
{
  "diagnostic_insight": {
    "zh": "您如龍行於天，氣勢充盈，然此刻陽剛過盛，恐有亢龍之憂。您的意志如磐石堅定，行動如疾風迅雷，卻忘了張弛有度之理。",
    "en": "You are like a dragon soaring in the sky, full of vigor..."
  }
}
```

**青少年版**（目標）：
```json
{
  "diagnostic_insight": {
    "zh": "你現在的能量就像火焰全開，燃燒得很旺。但你知道嗎？火燒得太猛，會把自己也燒乾。你衝勁十足，什麼都想做到最好，但有時候，慢下來不是輸，是為了跑更遠。",
    "en": "Your energy is like a flame at full blast, burning intensely. But you know what? A fire that burns too fiercely will consume itself. You're full of drive, wanting to excel at everything, but sometimes, slowing down isn't losing—it's about running farther.",
    "age_group": "teen"
  }
}
```

**調整重點**：
- ✅ 「您」→「你」
- ✅ 「龍行於天」→「火焰全開」（更現代的比喻）
- ✅ 「張弛有度」→「慢下來不是輸」（白話表達）
- ✅ 語氣更親近、不說教

---

### 1.2 活動形式調整

#### 調整原則

**原則 1：現代化活動**
- 增加運動（籃球、跑步、瑜伽）
- 增加創意（音樂、繪畫、寫作）
- 增加社交（社團、聚會、線上社群）

**原則 2：縮短時間**
- 從 30-60 分鐘 → 10-20 分鐘
- 符合青少年的專注時長
- 容易執行

**原則 3：數位元素**
- 融入 App（冥想 App、習慣追蹤）
- 社交媒體（限制使用時間）
- 遊戲化（挑戰、成就）

#### 範例對比

**成人版**（當前）：
```json
{
  "id": "exhaust_fire_to_earth_01",
  "title": "靜心書寫",
  "steps": [
    "準備筆墨與紙張",
    "每晚睡前 30 分鐘，寫下今日三件讓你感到「燥」的事",
    "寫完後，將紙張折疊，象徵「收納」",
    "連續 14 天"
  ],
  "duration": "30 分鐘"
}
```

**青少年版**（目標）：
```json
{
  "id": "exhaust_fire_to_earth_teen_01",
  "title": "能量日記（Vibe Journal）",
  "steps": [
    "打開手機或筆記本（用你喜歡的方式）",
    "每天睡前 10 分鐘，寫下今天三件讓你「覺得煩」的事",
    "寫完後，深呼吸三次，想像這些煩惱被收進一個盒子裡",
    "持續 7 天（一週挑戰）"
  ],
  "duration": "10 分鐘",
  "age_group": "teen",
  "gamification": "可設定 App 提醒，完成 7 天可獎勵自己"
}
```

**調整重點**：
- ✅ 時間：30 分鐘 → 10 分鐘
- ✅ 工具：筆墨 → 手機或筆記本（青少年熟悉）
- ✅ 語言：「燥」→「覺得煩」（白話）
- ✅ 週期：14 天 → 7 天（更短，易達成）
- ✅ 遊戲化：增加獎勵機制

## 🎯 任務 2：老年版本（60+ 歲）

### 2.1 語言風格調整

#### 調整原則

**原則 1：白話文優先**
- 減少文言隱喻
- 使用生活化表達
- 字體可放大（技術層面）

**原則 2：降低理解門檻**
- 簡化複雜概念
- 增加解釋說明
- 避免生僻字

**原則 3：尊重與理解**
- 語氣溫和、尊重
- 理解老年人的智慧
- 不居高臨下

#### 範例對比

**成人版**（當前）：
```json
{
  "diagnostic_insight": {
    "zh": "您如龍行於天，氣勢充盈，然此刻陽剛過盛，恐有亢龍之憂。",
    "en": "..."
  }
}
```

**老年版**（目標）：
```json
{
  "diagnostic_insight": {
    "zh": "您現在的狀態就像火燒得很旺。雖然有衝勁，但用力過度，反而容易累。就像燈油燒太快，雖然明亮，卻不持久。",
    "en": "Your current state is like a fire burning intensely. Though you have drive, overdoing it makes you tired easily. Like lamp oil burning too fast—bright but not lasting.",
    "age_group": "senior"
  }
}
```

**調整重點**：
- ✅ 「龍行於天」→「火燒得很旺」（白話比喻）
- ✅ 「陽剛過盛，恐有亢龍之憂」→「用力過度，反而容易累」
- ✅ 增加日常生活比喻（燈油）

### 2.2 活動形式調整

#### 調整原則

**原則 1：降低體力要求**
- 靜態活動優先（書法、閱讀、靜坐）
- 低強度活動（散步、太極、園藝）
- 考慮身體限制

**原則 2：日常化活動**
- 融入日常生活（洗碗、澆花、整理）
- 不需要特殊設備
- 在家即可完成

**原則 3：社交融合**
- 增加社交元素（與朋友聊天、參加社區活動）
- 減少孤獨感
- 增加互動性

#### 範例對比

**成人版**（當前）：
```json
{
  "id": "exhaust_fire_to_earth_01",
  "title": "靜心書寫",
  "steps": [
    "每晚睡前 30 分鐘，寫下今日三件讓你感到「燥」的事",
    "連續 14 天"
  ],
  "duration": "30 分鐘"
}
```

**老年版**（目標）：
```json
{
  "id": "exhaust_fire_to_earth_senior_01",
  "title": "晚安日記",
  "steps": [
    "每天晚上，花 15 分鐘寫下今天的心情（用筆或手機都可以）",
    "不用寫很多，三五句話就好",
    "寫完後，泡杯溫茶，靜靜坐一會兒",
    "持續一週，看看心情有沒有變化"
  ],
  "duration": "15 分鐘",
  "age_group": "senior",
  "notes": "可以用大字體筆記本，或請家人協助設定手機"
}
```

**調整重點**：
- ✅ 時間：30 分鐘 → 15 分鐘
- ✅ 語言：「燥」→「心情」（更白話）
- ✅ 步驟更詳細（「三五句話就好」）
- ✅ 增加舒適元素（泡茶、靜坐）
- ✅ 考慮數位障礙（提供協助建議）
