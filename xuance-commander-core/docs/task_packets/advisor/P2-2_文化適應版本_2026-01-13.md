### 2. 文化適應的必要性

#### 2.1 為什麼需要文化適應？

**問題 1：文化符號障礙**

**當前敘事**（強烈東方）：
```
您如龍行於天，氣勢充盈，然此刻陽剛過盛，恐有亢龍之憂。
```

**西方用戶反應**：
- 「龍？我們文化中龍是邪惡的」（文化衝突）
- 「陽剛？陰陽是什麼？」（概念陌生）
- 「亢龍之憂？什麼意思？」（語言障礙）

---

**問題 2：情境描述偏東亞**

**當前情境**（東亞文化）：
```
您與長輩的衝撞，如水拍巖石。長輩的期待，如山壓頂...
```

**西方用戶反應**：
- 「長輩期待？我們家很民主的」（家庭結構不同）
- 「為什麼要順從長輩？」（價值觀差異）

---

**問題 3：玄學概念不普及**

**當前概念**（需要文化背景）：
- 五行（木火土金水）
- 八卦（乾坤坎離震巽艮兌）
- 易經 64 卦
- 氣、勢、局

**西方用戶反應**：
- 「這些概念我完全不懂」（知識門檻）
- 「能不能用我能理解的方式說？」（理解困難）

---

#### 2.2 文化適應的價值

**價值 1：擴大市場**
- 從「東亞市場」→「全球市場」
- 市場規模擴大 5-10 倍

**價值 2：保持核心**
- 玄學語境不變（使用普世性隱喻）
- 診斷邏輯不變（Layer 1 科學內核）
- 只調整 Layer 2 的文化表達

**價值 3：真正達成「全人類」目標**
- 不再受限於特定文化
- 符合終極目標：All Humans, All Cultures

## 🎯 任務 1：文化中性版本

### 1.1 符號中性化

#### 調整原則

**原則 1：使用自然元素**
- 龍 → 風暴、火焰、洪流
- 鳳凰 → 重生、轉化
- 山 → 穩固、堅定
- 水 → 流動、適應

**原則 2：使用普世隱喻**
- 四季（春夏秋冬）
- 自然現象（風雨雷電）
- 日常經驗（燈火、道路、河流）

**原則 3：保持玄學語境**
- 仍使用能量、流動、平衡
- 仍使用元素（雖然不強調「五行」）
- 仍使用狀態描述（不用心理學詞彙）

#### 範例對比

**東方版**（當前）：
```json
{
  "diagnostic_insight": {
    "zh": "您如龍行於天，氣勢充盈，然此刻陽剛過盛，恐有亢龍之憂。您的意志如磐石堅定，行動如疾風迅雷，卻忘了張弛有度之理。",
    "en": "You are like a dragon soaring in the sky..."
  }
}
```

**文化中性版**（目標）：
```json
{
  "diagnostic_insight": {
    "zh": "你現在的能量就像一場風暴，力量強大且充滿氣勢。但風暴若不停歇，最終會耗盡自身。你的意志堅定如山，行動迅速如閃電，卻忘了即使是最強的風，也需要平靜的時刻。",
    "en": "Your energy is like a storm now—powerful and commanding. But a storm that never stops will eventually exhaust itself. Your will is firm as a mountain, your actions swift as lightning, yet you've forgotten that even the strongest wind needs moments of calm.",
    "version": "universal"
  }
}
```

**調整重點**：
- ✅ 「龍」→「風暴」（普世自然現象）
- ✅ 「陽剛過盛」→「力量強大」（去除陰陽概念）
- ✅ 「亢龍之憂」→「會耗盡自身」（直白表達）
- ✅ 保持玄學語境（能量、流動、平衡）

### 1.2 情境普世化

#### 調整原則

**原則 1：跨文化通用情境**
- 工作壓力（全球共通）
- 人際關係（家庭、朋友、伴侶）
- 自我價值（成就、認同）
- 時間管理（忙碌、失衡）

**原則 2：減少文化特定價值觀**
- 避免強調「孝道」「面子」「集體主義」
- 改用「尊重」「自尊」「社群」等普世概念

**原則 3：保持核心洞察**
- 診斷的核心洞察不變
- 只調整文化表達方式

#### 範例對比

**東方版**（當前）：
```json
{
  "context_fragments": {
    "authority": {
      "situation_clauses": [
        "您與長輩的衝撞，如水拍巖石。長輩的期待，如山壓頂，讓您喘不過氣..."
      ]
    }
  }
}
```

**文化中性版**（目標）：
```json
{

      "situation_clauses": [

      ]
    }
  }
}
```

**調整重點**：
- ✅ 「長輩」→「權威」（更普世）
- ✅ 保留自然隱喻（水、石、山）
- ✅ 減少「孝道」色彩

### 1.3 語言可理解化

#### 調整原則

**原則 1：減少文化專有名詞**
- 五行 → 元素（Element）
- 八卦 → 領域（Domain）
- 易經 → 古老智慧（Ancient Wisdom）

**原則 2：增加解釋性表達**
- 「氣」→「能量」（Energy）
- 「勢」→「趨勢」（Momentum）
- 「局」→「狀況」（Situation）

**原則 3：保持玄學本質**
- 不轉換為心理學語言
- 仍使用能量、流動、平衡等概念
- 只是用更通用的詞彙

#### 範例對比

**東方版**（當前）：
```json
{
  "action_guidance": {
    "zh": "此時當以柔克剛，學會收斂鋒芒。建議您採「金生水」之法，將過盛的剛健之氣，轉化為深沉的智慧儲藏。",
    "en": "..."
  }
}
```

**文化中性版**（目標）：
```json
{
  "action_guidance": {
    "zh": "此時需要柔軟勝過堅硬，學會收斂你的力量。建議你將強烈的能量，轉化為深層的智慧累積，就像將火的熱度，轉變為溫暖的光芒。",

    "version": "universal"
```

**調整重點**：
- ✅ 「金生水」→「將能量轉化」（去除五行專有名詞）
- ✅ 「剛健之氣」→「強烈的能量」（更通用）
- ✅ 保持玄學語境（能量、轉化、智慧）

- 可選顯示（用戶可選擇是否看註釋）
- 不打斷閱讀流暢性

#### 範例

**未註釋版**（當前）：
```json
{
  "diagnostic_insight": {
    "zh": "您的氣場呈現「木氣過旺」之象，像是春天的樹木瘋狂生長，卻根基不穩。",
    "en": "Your energy field shows 'excessive Wood Qi'..."
  }
}
```

**註釋版**（目標）：
```json
{
  "diagnostic_insight": {
    "zh": "您的氣場呈現「木氣過旺」之象，像是春天的樹木瘋狂生長，卻根基不穩。",
    "en": "Your energy field shows 'excessive Wood Qi'...",
    "annotations": {
      "wood_qi": {
        "zh": "「木」在五行中代表生長、擴張、行動的能量。木氣過旺意味著行動力強但缺乏穩定。",
        "en": "'Wood' in Five Elements represents growth, expansion, and action energy. Excessive Wood Qi means strong action but lacking stability."
      }
    }
  }
}
```

**調整重點**：
- ✅ 增加 annotations 欄位
- ✅ 解釋「木氣」概念
- ✅ 用白話說明
- ✅ 不影響原有敘事
