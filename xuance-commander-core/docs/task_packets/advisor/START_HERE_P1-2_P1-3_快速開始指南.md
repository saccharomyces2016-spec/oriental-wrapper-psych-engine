## 🎯 P1-2：動態敘事組合引擎

### 用一個例子理解

**通用敘事**（當前）：
```
您如龍行於天，氣勢充盈，然此刻陽剛過盛，恐有亢龍之憂。
您的意志如磐石堅定，行動如疾風迅雷，卻忘了張弛有度之理。
```
→ 所有人看到相同的敘事

**個人化敘事**（P1-2 目標）：
```
您如龍行於天，氣勢充盈，然此刻陽剛過盛，恐有亢龍之憂。
[針對 Authority 用戶插入]
您與權威的碰撞，並非來自外界的壓制，而是您內在的剛強無法順應天時。
長輩或導師的指引，您視為束縛，卻忘了真正的強者，懂得在必要時低頭。
```
→ 不同情境的用戶看到不同的敘事

**效果**：用戶會說「天啊，這說的就是我！」

### 你們要交付什麼？

**10 個檔案，分 3 個任務**：

**任務 1：標記插入點**（2 個檔案，3-4 天）
- 在 64 個卦象敘事中標記「哪裡可以插入個人化片段」

**任務 2：建立敘事片段庫**（7 個檔案，4-5 天）

- 例如：
  - Authority：「您與長輩的衝撞，如水拍巖石，激起千層浪花」
  - Resource：「您對資源的渴求，如乾涸之地渴望甘霖」
  - Peer：「與同儕的較量中，放下輸贏，才能看見更大的局」

**任務 3：組合規則**（1 個檔案，1 天）
- 定義「如何將片段插入卦象敘事」的規則

### 關鍵要求

✅ **必須**：
- 100% 玄學語境（使用：氣、勢、局、五行、陰陽）
- 情境對號入座（用戶感到「這說的就是我」）
- 語氣一致（插入片段與原敘事無縫融合）
- 中英雙語

❌ **禁止**：
- 心理學詞彙（焦慮、憂鬱、壓力）
- 醫療詞彙（治療、症狀、疾病）
- 空泛建議（多休息、放鬆心情）

## 🎯 P1-3：長鏈風險預測

### 用一個例子理解

**短鏈預測**（當前）：
```
chronic_alertness（慢性警覺）
→ 持續警覺 → 體力耗盡 → 慢性疲勞
```
→ 2-3 個階段，用戶可能覺得「還不夠嚴重」

**長鏈預測**（P1-3 目標）：
```
chronic_alertness（慢性警覺）- 職場崩潰路徑

Stage 1：持續警覺狀態
→ 對工作環境保持高度警戒，隨時準備應對突發狀況
→ 如弓弦繃緊，箭在弦上

Stage 2：體力與專注力下降
→ 長期高壓導致體力透支，注意力難以集中，小錯誤開始出現
→ 如燈油將盡，光芒搖曳
→ 警示：頻繁忘記重要事項、需要更多咖啡維持狀態

Stage 3：工作失誤累積與職業危機
→ 錯誤頻率上升，開始影響工作績效，上司或同事開始質疑能力
→ 如堤壩裂縫，水滴穿石
→ 警示：連續數週無法達成目標、收到負面評價或警告

Stage 4：職業崩盤與生活危機
→ 失去工作或被迫降職，經濟壓力驟增，家庭關係緊張，自我價值感崩塌
→ 如大廈傾頹，一瞬崩解
→ 後果：失去穩定收入、家庭衝突加劇、身心健康全面惡化
```
→ 4 個階段，讓用戶看見「遠期後果」

**效果**：用戶會說「我必須現在改變！」

### 你們要交付什麼？

**24-34 個檔案，分 3 個任務**：

**任務 1：選擇高風險 Facet**（1 個檔案，2 天）
- 從 30 個 Facet 中選擇 20-30 個需要長鏈預測的
- 分析每個 Facet 的「連鎖性」（會引發哪些領域的問題）

**任務 2：設計長鏈路徑**（21-31 個檔案，6-8 天）
- 為每個 Facet 建立 2-3 條長鏈路徑
- 每條路徑 3-4 個 stage
- 每條路徑至少 1 個分支點
- 例如：
  - 路徑 1：職場崩潰路徑（工作 → 績效 → 失業 → 經濟危機）
  - 路徑 2：健康崩潰路徑（警覺 → 失眠 → 疾病 → 生活癱瘓）

**任務 3：預防建議**（包含在任務 2 中）
- 為每個 stage 建立具體的預防建議
- 例如：
  - Stage 1：「建立工作與休息的明確邊界」
  - Stage 2：「主動與主管溝通工作負荷」
  - Stage 3：「暫停所有非必要工作，尋求專業協助」

### 關鍵要求

✅ **必須**：
- 100% 玄學語境
- 邏輯合理（每個 stage 是前一個的合理後果）
- 漸進惡化（Stage 1 → 2 → 3 → 4 逐步嚴重）
- 具體可感（使用具體情境，避免抽象描述）
- 中英雙語

❌ **禁止**：
- 心理學/醫療詞彙
- 恐嚇或威脅語氣（「你一定會...」）
- 空泛建議
