# Failure Patterns (Failure Distillation)

本文件萃取「東方玄學／命理系統」在歷史應用與現代化轉型中，反覆出現的失敗與崩潰模式。

## 1. 結構性崩潰（Structural Collapse）

### A. 參數爆炸與權重失衡（The Shen-Sha Overload）
- **來源**：傳統八字神煞派、紫微斗數全書（雜曜）。
- **現象**：系統定義了超過 100 種以上的「星／煞／神」，但缺乏統一的權重計算公式。
- **失敗點**：當「吉星」與「凶星」同時出現時（這是常態），系統無法給出「淨值（Net Value）」，導致解讀淪為自由心證或話術 cherry-picking。
- **後果**：使用者感到困惑，系統無法自動化運算。

### B. 運算精度脆折（Calculation Fragility）
- **來源**：紫微斗數（時辰邊界）、鐵板神數。
- **現象**：結構高度依賴「單一輸入值的絕對精確」。例如：出生時間差 1 分鐘（跨時辰），盤面結構 100% 翻轉，無過渡地帶。
- **失敗點**：現實世界數據（出生時間）常有誤差，系統缺乏模糊容錯（Fuzzy Logic）或邊緣平滑機制。
- **後果**：輸入稍有誤差，輸出完全不可信（False Precision）。

### C. 邏輯死鎖（The Self-Canceling Logic）
- **來源**：姓名學（三才五格 vs 八字喜用）。
- **現象**：不同子系統對同一符號的定義衝突。例如：姓名學數理認為「水」是吉（基於筆畫），但八字結構認為「水」是忌神（基於日主強弱）。
- **失敗點**：缺乏「元邏輯（Meta-logic）」來裁決子系統衝突。
- **後果**：多系統混用時，結論互相抵銷，導致「什麼都像、什麼都不準」。

## 2. 語境脫節（Context Drift）

### A. 階級標籤僵化（Fossilized Class Labels）
- **來源**：古籍斷語（如《三命通會》、《滴天髓》）。
- **現象**：使用農業社會的階級標籤（如：富、貴、貧、賤、僧、道、刑、剋）。
- **失敗點**：將「心理狀態／特質」強行綁定為「社會結果」。現代社會流動性高，同樣的特質可能在古代是「賤」（不穩定），在現代是「創業者」（靈活）。
- **後果**：用戶感到被冒犯或覺得荒謬。

### B. 性別偏見導致的結構失效（Gender Bias Failure）
- **來源**：傳統女命論斷（如「女命傷官」）。
- **現象**：將「強勢、主見、行動力」等特質，在女性身上定義為「剋夫／不幸」。
- **失敗點**：結構無法區分「特質本身」與「社會對特質的評價」。當社會價值觀改變，原有的「凶」結構在現代反而成為「成功」結構。
- **後果**：模型對現代女性用戶完全失效。

## 3. 互動性缺失（Interaction Vacuum）

### A. 單向宿命論（Determinism without Agency）
- **來源**：鐵板神數、傳統流年斷法。
- **現象**：輸出僅包含「事件預測」（e.g., "三十歲結婚"），不包含「變量」或「建議」。
- **失敗點**：使用者被視為靜態客體，系統未設計「使用者行為如何反饋影響結果」的路徑。
- **後果**：產生「習得性無助」或因預測未發生而導致系統信譽破產。
