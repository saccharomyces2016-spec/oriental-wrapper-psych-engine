## Accepted Risk-Chain Frameworks

### Framework A — Passive Pressure Accumulation
Core Idea:
- Income inflow stagnates while expense pressure increases.
- System compensates by compression and reserve extraction.

Bands:
- Mid: Pressure Injection Cycle : 氣滯流塞（中｜悶壓上涌）
  - Severity Cue: 中度｜可控但持續惡化｜先止漏再加速 : 氣滯流塞（中｜悶壓上涌）
  - Severity Cue: 中度｜可控但持續惡化｜先止漏再加速
- High: Reserve Draft Cycle : 基石掏空（高｜暗耗存糧）
  - Severity Cue: 高度｜已動到根基｜先止血再談翻身 : 基石掏空（高｜暗耗存糧）
  - Severity Cue: 高度｜已動到根基｜先止血再談翻身

Key Structural Signals:
- Repetitive monitoring without action
- Transition from flow usage to reserve usage
- Time-horizon collapse (weeks replace months)
