## Governance Rule
- Advisors MAY propose new metaphor domains.
- Proposals MUST include:
  - Domain definition
  - Example vocabulary
  - Boundary with existing domains
- Final inclusion requires Owner Decision.
- Rejected domains are NOT to be referenced in narrative output.

## Design Intent
This structure ensures:
- Perceived narrative depth for users
- Strict machine-checkable boundaries
- Zero impact on existing P0-3 conclusions
