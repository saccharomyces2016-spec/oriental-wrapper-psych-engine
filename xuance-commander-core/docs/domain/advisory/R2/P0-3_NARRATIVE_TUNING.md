# P0-3 Narrative Bite Tuning v1 (表層語感規格)

## Goal
在不越界（不給具體人生指令、不做醫療/法律/投資建議、不做具體事件預言）的前提下，提高「咬合感」。

## Technique 1: 去概率化斷語（斷其象，不卜其事）
- 移除：可能、也許、或許、好像
- 保留「對當下狀態的肯定」，禁止「對未來事件的斷言」
- 句式參考：
  - ✅「此刻之象已明：____。」
  - ✅「這不是錯覺，是局勢在回應你。」
  - ❌「你一定會____」（宿命/預言紅線）

## Technique 2: 肉身通感（以觸覺落地）
- 將抽象詞轉為可感的物理觸覺：濕、冷、熱、燥、重、黏、刺、悶
- 禁止使用現代心理學名詞（依 M3 禁用表）
- 句式參考：
  - ✅「氣息黏滯，胸口像壓著未乾的濕布。」
  - ✅「火氣上浮，皮膚發緊，夜裡難收神。」

## Technique 3: 表裡悖論（外觀如常、底層已變）
- 固定結構：
  - 「外人所見 A（整齊/平穩/如常），唯你自知 B（暗潮/腐變/滲漏）。」
- 目的：點破「表層可維持，底層在耗損」的結構矛盾
- 句式參考：
  - ✅「倉外堆疊齊整，倉底卻已微熱生霉。」
  - ✅「田埂仍在，地力卻早已透支。」

## Usage Limits
- 三招只用於 **L1–L3** 的狀態描寫，不得把「斷語」延伸成「具體指令」。
- L4 仍以「姿態/方位/修繕」為主，不落到生活操作指令。
