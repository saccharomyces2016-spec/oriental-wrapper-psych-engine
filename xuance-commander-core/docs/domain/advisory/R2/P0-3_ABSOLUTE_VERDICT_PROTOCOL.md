# P0-3 Absolute Verdict Protocol (R2)

Role: R2 (Esoteric Narrative Refiner)
Status: DRAFT-FROZEN (Narrative-Safe, Adjustable Later)
Scope: P0-3 Narrative Sharpness
Primary Constraint: P0-2 Output Contract

---

## 1. Scope of Absolute Verdicts（肯定句使用邊界）

### Allowed（必須使用肯定句）
- 主觀體感、能量流向、結構現狀、內外不一致
- 定義：Empathic Certainty（共情式確信）

Examples:
- 「你的火燒得太旺了。」
- 「你的倉底已經見空。」
- 「這是一種無水的燃燒。」

### Forbidden（禁止肯定句）
- 客觀事實/數字/具名人物
- 未來結果/保證
- 醫療或心理診斷
- 道德評斷/歸責

Examples:
- 「你明年一定會發財。」
- 「你的上司是小人。」
- 「你有重度憂鬱症。」

---

## 2. Absolute Verdict Guardrail（護欄）

- Allowed: 「你此刻感到窒息。」（狀態）
- Forbidden: 「你的環境讓你窒息。」（外部指責）
- Allowed: 「你的收成追不上消耗。」（流向）
- Forbidden: 「你的薪水太低了。」（數字事實）

---

## 3. Sharp Non-Fatalistic Templates（銳利但不宿命）

### A. Structural Reveal（結構揭露）
Pattern: 「這並非 {表層原因}，而是 {結構真相}。」
Example: 「這並非你不夠勤懇，而是你的田地本身就位於旱區。」

### B. Hidden Cost（代價清算）
Pattern: 「你以為自己是在 {正面行為}，其實是在 {隱形消耗}。」
Example: 「你以為自己是在守護倉廩，其實是在用肉身堵住堤壩的缺口。」

### C. Seasonal Mismatch（時節錯位）
Pattern: 「天候已轉至 {季節}，而你卻還穿著 {錯誤裝備}。」
Example: 「天候已轉至嚴冬，而你卻還穿著單薄的夏衣在田間奔走。」

---

## 4. Enforcement Rules（執行規範）
- 僅限 L1–L3 使用
- 不得預測、不下指令
- 任何結果/未來暗示 → 無效

Binding for all P0-3 narratives.
