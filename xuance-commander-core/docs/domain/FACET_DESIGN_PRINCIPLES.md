# Facet Design Principles（主線硬規範）

## 1. 東方玄學語境（Mandatory）
所有 Facet 題型與結果展示，必須建立在東方玄學語境之上。
允許混用但需保持內在一致性：

- 五行（金木水火土）
- 陰陽兩極
- 八卦（先天 / 後天）
- 天干地支
- 三才 / 五格 / 81 象

禁止：
- 心理測驗語言
- 西方人格量表敘事
- 解釋推演原理給使用者

## 2. UI / 敘事層使用原則
- 題目語言：隱性象徵，不教學
- 結果呈現：盤面、走勢、狀態
- 命名避免「好 / 壞」，使用「勢 / 位 / 壓力」

## 3. 可回調規則（Hard）
在 Golden Tests 封板前，以下皆視為可調整：
- 題目文字
- 分數區間
- band 命名
- 風險鏈敘事

## 4. 顧問升級義務
當出現以下情況，必須主動升級顧問角色：
- 玄學體系混用是否衝突
- 推演邏輯是否自洽
- 文化象徵是否誤導

最終裁決權屬於指揮官。
