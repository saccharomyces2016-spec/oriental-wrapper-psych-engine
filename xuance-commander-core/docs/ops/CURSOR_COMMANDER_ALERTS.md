# CURSOR_COMMANDER_ALERTS（Cursor 對總指揮 GPT 提醒事項）

## 目的
記錄 Cursor（總工程師）發現的需要總指揮 GPT 關注、決策或協助處理的事項。

## 使用規則
- **優先級標記**：P0（立即處理）、P1（建議處理）、P2（可選處理）
- **狀態標記**：OPEN（待處理）、IN_PROGRESS（處理中）、RESOLVED（已解決）、DEFERRED（延後）
- **更新規則**：事項處理完成後，狀態改為 RESOLVED，並記錄處理結果
- **清理規則**：RESOLVED 狀態的事項保留 7 天後移除（或移到歷史記錄）
