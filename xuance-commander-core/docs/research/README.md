# Research Notes（內部研究層｜不可直接外露）

用途：
- 放「內部研究總稿 / 世界觀推導 / 方法論筆記」。
- 內容可包含心理學/神經科學/工程推導等「內核語言」。

硬規則：
- ✅ 只能用來「推導」domain 的題目/計分/敘事/風險鏈。
- ❌ 不得把本層文字直接貼進任何使用者可見輸出（questions/narratives/recommendations/riskchains）。

正確流程：
1) 指揮官先從本層抽出「可用訊號/推導骨架」
2) 產出給顧問的短 briefing（docs/gem/briefs/）
3) 顧問輸出（docs/gem/runs/）→ 指揮官審核 → 才能落盤到 domain/
