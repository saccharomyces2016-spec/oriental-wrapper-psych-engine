# 「記得寫進去」= 直接產生指令包（硬規則｜SSOT）

定義：
- 使用者對指揮官（GPT）說「記得寫進去」時，
  代表：**立即產出可重跑（idempotent）的指令包**，把內容寫入 SSOT 檔案。

硬規則：
1) 禁止把規則/結論直接手動寫進 MIN/FULL 快照檔。
2) 必須寫入專屬 SSOT 檔案（governance / process / domain / docs 等）。
3) 完成後必須重建快照（AUTO），讓 MIN/FULL 提取該變更的指標與指針。
4) 若需修改既有制度：以「新增區塊」或「受標記區塊」方式更新，避免漂移。

驗收：
- SSOT 檔案存在且內容完整
- CURRENT 有新增指標/指針
- MASTER_MIN 重建後能看到變更摘要（僅指標/指針）
