# 第一章｜系統總體設計哲學

## 1.1 核心目標

DESTINY CODE 的目標不是「測驗分數」，而是：

> 用最少題目，在最短時間內，  
> **收斂出一個人正在面對的局、當下的狀態、內在的結構，以及氣數的盈虧。**

本系統同時滿足：
- 心理測量的鑑別度
- 命理占斷的權威感
- 使用者體驗的沉浸與儀式性
- AI 分析的可結構化輸入

## Round 1｜命題入口（Domain Selection）

### 設計原則
- 一字一域，建立「命理占斷」的第一印象
- 避免心理學分類語感
- 不問狀態、不問原因，只問「此刻命運落點」

### 題目
> **此刻，你最想解的，是哪一個命題？**

### 選項（8 Domains）
- **緣**｜感情、關係、牽絆、因果
- **財**｜金錢、安全感、資源流動
- **身**｜健康、能量、身體狀態
- **業**｜工作、學業、責任、定位
- **家**｜家庭、照顧、角色、歸屬
- **局**｜轉折、變局、困局、抉擇
- **心**｜情緒、內在、感受、心理
- **命**｜整體狀態、人生方向、運勢

> Round 1 僅做「領域路由」，不做評分。

## Round 2｜四象狀態（氣・勢・界・動）

### 設計原則
- 每個 Domain 固定四象：
  - **氣**：內在狀態
  - **勢**：外在壓力
  - **界**：邊界／混亂
  - **動**：應對策略
- 採用 **東方玄學意象 + 體感錨點**
- 前台給人看，後台給 AI 解碼（Decoder Ring）

---

### Domain｜緣

**Prompt**  
你立於緣份的流年渡口，霧氣漸起，哪一幕最貼近你此刻的身體感受？

- **氣**：春寒未退，你覺得**胸口鬱結**，像含苞之花低垂，想開口卻發不出聲。
- **勢**：狂風驟起，舊事如重拳拍擊，你不自覺**縮起肩背**，只能承受。
- **界**：隔著濕霧看人，忽遠忽近，怎麼用力都**對不上焦**，一陣暈眩。
- **動**：夜色之中，你**壓低氣息**，悄悄解纜，讓渡船順水退去。

---

### Domain｜財

**Prompt**  
站在財庫之前，看著水脈流動，你身體的第一個反應是？

- **氣**：守著一口舊井，**喉頭乾澀**，反覆張望，唯恐乾涸。
- **勢**：洪水逼近，你**呼吸急促**，胸口像壓著大石。
- **界**：田畝交錯，你**腳底發虛**，分不清哪塊能站穩。
- **動**：你踏入水中，忍受**刺骨涼意**，一步步試探新源。

---

### Domain｜身

**Prompt**  
閉目內觀，感受丹田氣場，此刻最像哪一象？

- **氣**：爐中餘燼尚溫，你卻覺得**四肢沈重**，提不起力氣。
- **勢**：烈火竄行，你感到**燥熱難安**，停不下來。
- **界**：城牆剝落，一陣**虛寒鑽入**，卻無暇修補。
- **動**：你關上門窗，**長吐一口氣**，讓肩頸慢慢鬆沉。

---

### Domain｜業

**Prompt**  
抬頭看向事業宮位，身體先給你的訊號是？

- **氣**：雲遮日光，你**胸悶氣短**，想動卻被黏住。
- **勢**：巨石壓道，你**咬緊牙關**，背脊發硬。
- **界**：路標林立，你**思緒斷片**，找不到焦點。
- **動**：你停下來調息，等**心跳回穩**再前行。

---

### Domain｜家

**Prompt**  
站在家宅之中，你最真實的身體反應是？

- **氣**：屋梁低垂，你**頸椎僵硬**，不敢抬頭。
- **勢**：氣流紊亂，你**腹部緊縮**，神經緊繃。
- **界**：房門常開，你感到**精氣外洩**，一陣虛脫。
- **動**：你**屏氣凝神**，先把混亂攬在自己身上。

---

### Domain｜局

**Prompt**  
站在命運交會點，你身體更接近哪種狀態？

- **氣**：舊殼未褪，你**皮膚緊繃**，動彈不得。
- **勢**：雷聲壓頂，你**脈搏狂跳**，不能再拖。
- **界**：岔路重疊，你**眼花撩亂**，分不清方向。
- **動**：你深吸一口氣，**挺起胸膛**，跨出那一步。

---

### Domain｜心

**Prompt**  
低頭看向心湖，你的身體感受是？

- **氣**：湖面死寂，你**知覺遲鈍**，情緒微弱。
- **勢**：暗流翻湧，你**胸口翻攪**，忽上忽下。
- **界**：倒影破碎，你**頭重腳輕**，分不清自己。
- **動**：你靜坐調息，讓混濁**慢慢落底**。

---

### Domain｜命

**Prompt**  
回望人生全局，你最誠實的反應是？

- **氣**：燈火微弱，你**眼皮沈重**，靠意志撐著。
- **勢**：大運推動，你**雙腳懸空**，身不由己。
- **界**：新舊交疊，你感到**時空錯置**。
- **動**：你**站穩腳跟**，準備重新落筆。

## Round 3｜結構抉擇（Ipsative）

### 設計原則
- 強迫二選一（Forced Choice）
- 問「命理法則」，不問情緒
- 文案風格：**大道／因果／氣數**

### Q1｜守中 vs 順勢
> 當大勢混沌，本心與外界背離，你信什麼？

- **A｜守中**：信本心，寧逆勢獨行。
- **B｜順勢**：信流轉，隨勢而通。

### Q2｜護根 vs 破局
> 命盤行至變爻，你如何應對？

- **A｜護根**：守住基業，不為未知豪賭。
- **B｜破局**：打碎舊局，求涅槃重生。

### Q3｜存真 vs 圓融
> 當因果需燃燒元神來換圓滿，你答應嗎？

- **A｜存真**：保全本命，不入孽緣。
- **B｜圓融**：以身渡人，修至圓滿。

## Round 4｜定錨強度（Intensity）

### 設計原則
- 僅 1 題
- 判定「氣數盈虧」
- 不新增結構，只校正強度

### 四組定錨

#### A｜守中型
- **平衡**：心如古井，定力尚存。
- **過載**：油盡燈枯，只剩硬撐。

#### B｜順勢型
- **平衡**：行雲流水，借力而行。
- **過載**：斷根浮萍，無處靠岸。

#### C｜護根型
- **平衡**：深溝高壘，基業安穩。
- **過載**：坐困愁城，城將崩塌。

#### D｜破局型
- **平衡**：壯士斷腕，破而後立。
- **過載**：萬丈深淵，懸空墜落。
