# ONTOLOGY & PATTERN TAG GOVERNANCE
Status: DRAFT (Skeleton)
Authority: Under WORLDVIEW_CANON_vNEXT_v1.md

## 1. Purpose
Define which **structural patterns** are:
- allowed for internal computation
- allowed for narrative exposure
- permanently forbidden from user-facing output

## 2. Tag Classes
- Structural Tags (internal-only)
- Transitional Tags (computable, non-exposed)
- Exposable Ontology Tags (limited, audited)

## 3. Exposure Rules
- No tag may imply diagnosis
- No tag may imply correctness or prescription
- No tag may collapse ambiguity

## 4. Hard Bans
- Psychological category tags
- Medical or mental-health analogues
- Deterministic fate claims

## 5. Audit Requirement
All exposed tags MUST be traceable to:
Structural Signal → Element Prior → Narrative Binding
